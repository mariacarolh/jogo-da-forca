Um **jogo da forca desenvolvido em C# para terminal/console**, criado como parte de um **trabalho acadêmico na faculdade**.

---

## Sobre o projeto

Este jogo permite que o usuário jogue o clássico **Jogo da Forca** diretamente no console.

O sistema suporta:

* 🎮 **Modo Single Player** com palavras carregadas de um arquivo externo
* 👥 **Modo Multiplayer**, onde um jogador define a palavra
* ❤️ Sistema de **vidas representadas por corações**
* 🔤 Histórico de **letras já utilizadas**
* 🎨 Uso de **cores no console** para melhorar a experiência visual
* 📂 Carregamento de palavras a partir de um **arquivo `.txt`**

---

## Objetivo acadêmico

Este projeto foi desenvolvido como parte de um **trabalho da faculdade**, com o objetivo de aplicar na prática conceitos como:

* Estrutura de repetição (`while`, `for`)
* Estrutura condicional (`if`, `switch`)
* Manipulação de **List<T>**
* Leitura de arquivos com **System.IO**
* Organização do código em **métodos**
* Interação com o usuário via **Console**

---

## Como executar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/jogo-da-forca.git
```

2. Abra o projeto no **Visual Studio**

3. Compile e execute o projeto (`F5`)
